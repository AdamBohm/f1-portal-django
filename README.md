# F1 Portal - Školní projekt Django

Webová aplikace vytvořená ve frameworku Django zaměřená na evidenci stájí, jezdců a kalendáře závodů Formule 1. Projekt splňuje požadavky na relační databázi (vztahy 1:N) a responzivní frontend.

**Autor:** Adam Bőhm

## 🏁 Funkce aplikace
- Úvodní stránka s dynamickými statistikami a kalendářem závodů.
- Seznam všech aktivních stájí F1 s barevným odlišením.
- Detailní profil stáje s výpisem přiřazených jezdců, jejich národností a automatickým počítadlem vítězství a nejrychlejších kol.
- Administrační rozhraní s validacemi vstupních dat (např. HEX kódy, startovní čísla 1-99).

## ⚙️ Použité technologie
- **Backend:** Python, Django, SQLite3
- **Frontend:** HTML5, Bootstrap 5, vlastní CSS
